TACS.NET Manager Help

This document describes the basic functions of the TACS.NET Manager desktop application. TACS.NET is a client/server framework designed to provide access control services for web and desktop applications built on Microsoft's .NET Framework.

Managing Projects

Projects represent individual database instances for a given application. Each project requires its own database connection string. When a user logs into the application with their TACS.NET credentials, the session is given the database connection string based on which project the user logs into. Users must be given access to individual projects, allowing you to manage database access on a per project basis.
